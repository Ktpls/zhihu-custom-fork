# 知乎样式修改器

使用方法：在页面左侧鼠标触摸出现眼镜按钮点击出现修改器弹窗<br/>
<br/>
一键极简模式，去除不必要的元素，给你最简单的知乎。<br/>
首页问题分类 tab，发现 tab 和个人中心可悬浮自定义位置。<br/>
支持版心修改，页面模块位置调整、隐藏，页面表头和图标修改<br/>
页面背景色修改，黑色为夜间模式<br/>
列表添加视频、回答和文章的 TAG<br/>
自动去除广告<br/>
更多功能请在插件里体验<br/>
想要的最终功能是页面大部分模块全可配置，目前努力更新中...<br/>
